    Projet de manipulation de données

    Language: Python

    Répo GitHub

    Ce projet a eu lieu lors de mon module de manipulation de données en Python.
    L'objectif de ce projet était de réaliser une api afin de générer des données aléatoirement.
    Notre contexte était la gestion de salle de jeu de différentes où des joueurs pourraient les rejoindre, ainsi que de prendre en compte le fait qu'une salle peut être pleine, que les joueurs peuvent sortir des salles et de créer de nouvelle si il n'y a plus de places disponibles.
    Pour ce projet j'étais accompagné par deux camarades de classe.

    Création d'une application liée à la moto

    Language: React native / JavaScript

    Répo GitHub

    Ce projet était notre sujet de projet développement. L'idée vient d'un camarade avec qui j'ai collaboré.
    L'idée était de mettre en place une application mobile afin d'aider les motards.es à se regrouper pour réaliser des trajets entre eux. Ce projet devait être réalisz en binôme, mais souhaitant réaliser le plus de fonctionnalités possible nous nous sommes regroupé en deux duos: un chargé de réaliser la partie front de l'application et l'autre de s'occuper de toute la partie back (binôme dont je faisais partie).
    Nous avons pu réaliser toutes les fonctionnalités principales. Les deux parties n'ont pas été reliées entre elles car ce n'étais pas nécessaire pour notre notation mais ce projet à pour but d'être continuer dans le futur.
